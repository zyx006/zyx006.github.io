Mac键盘符号和快捷键说明

发布于 2020-11-14 | 4分钟 | 833字数 | 浏览量：

本文迁移自原生Github Pages

⌘ Command
⇧ Shift
⇪ CapsLock
⌥ Option
⌃ Control

↩ Return/Enter
⌫ Delete
⌦ 向前删除键(Fn + Delete)
↑ 上箭头
↓ 下箭头
← 左箭头
→ 右箭头
⇞ PageUp(Fn + ↑)
⇟ PageDown(Fn + ↓)
Home  Fn + ←
End  Fn + →
⇥ 右制表符(Tpb键)
⇤ 左制表符(Shift + Tpb)
⎋ Escape(Esc)

Search/Replace(查询/替换)

Double Shift(⇧) 查询任何东西
Command(⌘) + F 文件内查找
Command(⌘) + G 查找模式下，向下查找
Command(⌘) + Shift(⇧) + G 查找模式下，向上查找
Command(⌘) + R 文件内替换
Command(⌘) + Shift(⇧) + F 全局查找(根据路径)
Command(⌘) + Shift(⇧) + R 全局替换(根据路径)
Command(⌘) + Shift(⇧) + S 查询结构(Ultimate Edition 版专用，需要在Keymap中设置)
Command(⌘) + Shift(⇧) + M 替换结构(Ultimate Edition 版专用，需要在Keymap中设置)

Usage Search(使用查询)

Option(⌥) + F7 / Command(⌘) + F7 在文件中查找用法 / 在类中查找用法
Command(⌘) + Shift(⇧) + F7 在文件中突出显示的用法
Command(⌘) + Option(⌥) + F7 显示用法

Compile and Run(编译和运行)

Command(⌘) + F 编译Project
Command(⌘) + Shift(⇧) + F9 编译选择的文件、包或模块
Control(⌃) + Option(⌥) + R 弹出 Run 的可选择菜单
Control(⌃) + Option(⌥) + D 弹出 Debug 的可选择菜单
Control(⌃) + R 运行
Control(⌃) + D 调试
Control(⌃) + Shift(⇧) + R, Control(⌃) + Shift(⇧) + D 从编辑器运行上下文环境配置

通用

Command(⌘) + C 复制
Command(⌘) + V 粘贴

   剪切

   Step 1:Command(⌘) + C 复制
   Step 2:在需要转移的文件夹位置使用Command(⌘) + Option(⌥) + V。(如果操作后悔了，还可以 Command(⌘) + Z 撤销。)

Command(⌘) + A 全选
Command(⌘) + S 保存
Command(⌘) + F 查找(Find)
Command(⌘) + M 最小化窗口
Command(⌘) + N 新建文件
Command(⌘) + O 打开
Command(⌘) + H 隐藏窗口
Command(⌘) + Shift(⇧) + S 另存
Command(⌘) + W 关闭
Command(⌘) + Q 退出
Command(⌘) + Z 撤销(如果是safari浏览器，就是恢复刚关闭的页面)
Command(⌘) + Shift(⇧) + Z 重做，也就是撤销的逆向操作
Command(⌘) + Option(⌥) + Esc 强制退出程序(类似于window下的任务管理器)
自带的输入法(Capslock 短按中英文切换，长按英文大写)
Command(⌘) + Shift(⇧) + 4 截取所选屏幕区域到一个文件，或按空格键仅捕捉一个窗口
Command(⌘) + Delete 将文件移至废纸篓(Windows键盘是 Win + Backspace)
Command(⌘) + Shift(⇧) + Delete 清倒废纸篓(Windows键盘是 Win + Shift + Backspace)
Space 快速查看选中的文件，也就是预览功能
Command(⌘) + ↑ 打开包含当前文件夹的文件夹，相当于Windows里的“向上”
Command(⌘) + Tab 在应用程序间切换(应用处于打开且在桌面的情况)

浏览器

Command(⌘) + L 光标直接跳至地址栏(可能被Windows响应锁屏)
Command(⌘) + '+' / '=' 放大页面
Command(⌘) + '-' 缩小页面
Control(⌃) + Tab 转向下一个标签页
Control(⌃) + Shift(⇧) + Tab 转向上一个标签页
Command(⌘) + R 刷新
Command(⌘) + T 新建一个选项卡
PageDown 向下滚动屏幕
PageUp 向上滚动屏幕
按下箭头键 向上、向下、向左或向右滚动
按下箭头键时按住Option(⌥)键大幅度滚动
enter即回车 重命名
Command(⌘) + 鼠标点击 多选不连续的文件或文件夹

其他

选中多个连续的文件或文件夹
按住Shift(⇧)键不放，使用鼠标依次点击(或按住鼠标后拖动)希望选择的文件或者文件夹即可
文件可以右键重命名
如果桌面的文件不对齐排列，可以右键整理
Shift(⇧) + Command(⌘) + C 打开“电脑”窗口
文件右键有一个“制作替身”，相当于新建一个快捷方式
